Add new place

You can find this page by navigating to Stay&Eat and then pressing the Add place button The page looks like this:

  1. Name input field
  2. Address input field
  3. Link input field
  4. Type selection
  5. Create button
  6. Close button

To successfully create an accommodation/restaurant you must fill in all fields. The minimum number of characters for name and address is three (3). The link must be in the appropriate format. And then choose which type of place you want to add. If all entries are correct, you will be able to click the Create button and create a place. Click on the Close button (x) to close the window and return to Stay&Eat page. But beware all previous changes will be deleted!!!

List of shortcuts

  • Move to the next field: Tab
  • Login (when a field is in focus): Enter
