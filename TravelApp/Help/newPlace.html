Add or Edit place

You can find this page by navigating to Stay&Eat and then pressing the Add place button
Or for editing you can press the last button in column of table and pick place for editing. The pages looks like this:

  1. Name input field
  2. Address input field
  3. Link input field
  4. Type selection
  5. Create (Edit) button
  6. Close button

To successfully create or edit an accommodation/restaurant you must fill in all fields. The minimum number of characters for name and address is three (3). The link must be in the appropriate format. And then choose which type of place you want to add. If all entries are correct, you will be able to click the Create or Edit button and create/edit a place. Click on the Close button (x) to close the window and return to Stay&Eat page. But beware all previous changes will be deleted!!!

List of shortcuts

  • Move to the next field: Tab
  • Login (when a field is in focus): Enter
  • Leave the page: Esc
