Sold trips page

If you want to see a list of all sold trips, you must first navigate to the page 'Sold Trips'.

The page that will be displayed contains the following elements:

  1. Month selection field
  2. Trip selection table with search
  3. Table of trips sold
  4. Report table

- By selecting a month, you select the month for which the sold trips will be shown.
- From the trip selection tables, you can select which trips you want to view sales for.
- In the sold trips table, the sold trips are displayed depending on the previously selected criteria. Each row in a table can be sorted.
- Finally, we can see the total price of the trips sold, as well as the number of trips sold.
