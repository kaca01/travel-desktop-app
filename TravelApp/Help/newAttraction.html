Add or Edit an attraction

You can find this page by navigating to All Attractions and then pressing the Add attraction button
Or you can find Edit button on which card of attraction. The page looks like this:

  1. Name input field
  2. Address input field
  3. Map
  4. Description
  5. Add photo button
  6. Save button
  7. Close button

To successfully create or edit an attraction you must fill in all fields. The minimum number of characters for name and address is three (3). The maximum number of characters for name is fifty (50). Also the start date must be before the end date. By pressing the Add photo button you can import an image from your computer. If all entries are correct, you will be able to click the Save button and create or edit a trip. Click on the Close button (x) to close the window and return to All attractions page. But beware all previous changes will be deleted!!!

Work with map

By clicking on the map you can set a pin. You can move the pin around the map. But you can also enter the location from the field.

List of shortcuts

  • Move to the next field: Tab
  • Login (when a field is in focus): Enter
