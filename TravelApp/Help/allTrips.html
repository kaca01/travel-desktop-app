All trips

After successful login you can see this page

The page displays the following elements:

  1. Search field
  2. Sort type selection field
  3. Trip display card

- A maximum of 50 characters can be entered in the search field.
- We can specify the sorting criteria using the selection field
- On the trip display card, we can see basic information about trip such as: name, image, description, price. Also at the bottom of the card there is a See Details button which, when clicked, opens a page for displaying details about the selected trip.
