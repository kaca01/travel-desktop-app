Trip details

If you want to see a details for selected trip. Press See details button on card of trip.

The page that will be displayed contains the following elements:

  1. Information about trip
  2. Description about trip
  1. Table of restaurants and accomodation for selected trip
  1. List of attraction for selected trip
  1. Details on map (restaurants, accomodations and attractions)
  1. Button for booking
  2. Button for buy a trip

List of shortcuts

  • Scroll left on navigation: Left arrow
  • Scroll right on navigation: Right arrow
  • Choose action: Enter
  • Back to All trips page: Escape
