Stay and eat table

If you want to see a list of all available restaurants and accommodation, you must first navigate to the page 'Stay&Eat'.

The page that will be displayed contains the following elements:

  1. Search field
  2. Add place button
  3. Table of restaurants and accomodation

- A maximum of 50 characters can be entered in the search field.
- Each row in the table can be sorted.
- The table shows a list of all restaurants and accomodatios.
- Clicking on the link see more will take you to the official page of the selected restaurant/accommodation
- The penultimate row (blue button) of the table represents the delete button. Clicking on that button deletes the restaurant/accommodation of the row in which the button was pressed.
- To delete the restaurant/accommodation you will need to confirm your action.
- The penultimate row (blue button) of the table represents the edit button. Clicking on that button edits the restaurant/accommodation of the row in which the button was pressed.
- To edit the restaurant/accommodation you will need to confirm your action.

List of shortcuts

  • Scroll left on navigation: Left arrow
  • Scroll right on navigation: Right arrow
  • Choose action: Enter
