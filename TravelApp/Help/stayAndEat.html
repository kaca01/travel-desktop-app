Stay and eat table

If you want to see a list of all available restaurants and accommodation, you must first navigate to the page 'Stay&Eat'.

The page contains a table in which the name, address, type, link are defined for each restaurant/accommodation

- Also on the page there is a Search field that enables searching by table
- Each row in the table can be sorted
- Clicking on the link see more will take you to the official page of the selected restaurant/accommodation
