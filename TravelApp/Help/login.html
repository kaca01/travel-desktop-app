Login on system

Login is the first page the user see when starting the application. The page looks like this:

  1. Email input field
  2. Password input field
  3. Button for login
  4. Button to go to the registration page

For a successful login, you need to enter the credentials of an existing user profile in the appropriate fields. After that, by clicking on the login button or by pressing the Enter, login to the system is performed. After a successful login, the user is redirected to the view that contains the functionalities intended for the type of registered user. In case the user does not have an existing profile, clicking on the button 'Sign up' opens it registration page where he can create a new profile.
