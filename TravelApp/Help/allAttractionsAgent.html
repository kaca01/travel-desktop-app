All attractions

If you want to view all available attractions, you must first navigate to the page 'Attractions'.

The page displays the following elements:

  1. Search field
  2. Sort type selection field
  3. Add attraction button
  4. Attraction display card

- A maximum of 50 characters can be entered in the search field.
- We can specify the sorting criteria using the selection field.
- By pressing the Add attraction button, you go to the form for adding a attraction.
- On the attraction display card, we can see basic information about attraction such as: name, image, description. There is also a Delete button. By pressing it and then confirming that action, you can delete the trip.
