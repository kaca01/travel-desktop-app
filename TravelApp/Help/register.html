Registration of a new profile

User profile registration can be accessed from the initial view of the application which shows the login to the system. The registration page looks like this:

  1. Name input field
  2. Surname input field
  3. Email input field
  4. Password input field
  5. Password re-entry (confirmation) field
  6. Sign up button
  7. Button to go to the login page

All fields on this page must be filled in for successful registration. Email must be unique at the system level. Name and surname must have at least three (3) characters. The password must be longer than eight (8) characters and an identical password must be found in the password re-entry (confirmation) field as in the password input field. The button will be inactive until the user enters the data correctly. By clicking on the 'Sign up' button or by pressing Enter registration is in progress. After successful registration, the user is informed about it and is transferred to the login page.

In case the user wants to login to an existing profile, click on the button 'Login' will take him to the login page.

List of shortcuts

  • Move to the next field: Tab
  • Login (when a field is in focus): Enter
