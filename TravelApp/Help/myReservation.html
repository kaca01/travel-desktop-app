My reservations table

If you want to see a list of all your own reserved trips, you must first navigate to the page 'Reservations'.

The page that will be displayed contains the following elements:

  1. Search field
  2. Reservation table
  3. Button to buy a trip
  4. Button to call off a reservation

- A maximum of 50 characters can be entered in the search field.
- Each row in the table can be sorted.
- The table shows a list of all your own reserved trips.
- By selecting a row in the table and clicking the buy button, you can buy a trip.
- By selecting a row in the table and clicking the call off button, you can call off a reservation.
- If there is no data in the table the buttons will be disabled

List of shortcuts

  • Scroll left on navigation: Left arrow
  • Scroll right on navigation: Right arrow
  • Choose action: Enter
