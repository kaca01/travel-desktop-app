Add or Edit trip

You can find this page by navigating to All Trips and then pressing the Add trip button
For Edit go to details about trip and press Edit button on top right corner
Pages looks identically The page looks like this:

  1. Name input field
  2. Starting location input field
  3. Ending location input field
  4. Starting date input field
  5. Ending date input field
  6. Price input field
  7. Description input field
  8. Add photo button
  9. Attraction selection
  10. Accomodations selection
  11. Restaurants selection
  12. Save button
  13. Close button

To successfully create or edit a trip you must fill in all fields. The minimum number of characters for name, starting and ending location is three (3). The maximum number of characters for name is fifty (50). Dates cannot be in the past. Also the start date must be before the end date. By pressing the Add photo button you can import an image from your computer. You can assign existing attractions, accommodations and restaurants to the new trip in the selection fields. If all entries are correct, you will be able to click the Create or Edit button and create/edit a trip. Click on the Close button (x) to close the window and return to All trips page. But beware all previous changes will be deleted!!!

List of shortcuts

  • Move to the next field: Tab
  • Login (when a field is in focus): Enter
  • Leave the page: Esc
